# 🏗️ Unit 01: Project Setup

Welcome to the first unit! In this section, we focus on the foundational configuration of an ASP.NET Core 10 application.

## 🎯 Learning Objectives
- **SDK Verification**: Confirming the .NET 10 environment.
- **Project Structure**: Understanding the files generated by the `mvc` template.
- **Middleware Pipeline**: Exploring how requests flow through the application.

## 📐 Architecture & Design
Even in a simple setup, we maintain:
- **Clean Configuration**: Separation of app settings and environment variables.
- **SOLID**: Each middleware has a Single Responsibility.

## 🛡️ Quality Standards
- **Health Checks**: Always ensure the application starts and responds.
- **Kdoc Documentation**: Every configuration block is documented.

## 🚀 How to Run
```bash
dotnet run
```
