# 🚀 Unit 01: Project Setup

Welcome to the first unit of the C# ASP.NET Core 10 fundamentals course. This unit focuses on setting up a solid, professional foundation for web development.

## 🎯 Learning Objectives
- [x] Understand the **.NET 10 project structure**.
- [x] Master the **Middleware Pipeline** in `Program.cs`.
- [x] Establish **Integration Testing** using `WebApplicationFactory`.
- [x] Configure **CI/CD Quality Gates** from day one.
