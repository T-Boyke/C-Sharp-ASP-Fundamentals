# 🚀 Unit 03: RSVP Application

This unit implements a complete **RSVP Application**, based on the classic example from Adam Freeman's *Pro ASP.NET Core MVC*. It demonstrates fundamental web development patterns in ASP.NET Core 10.

## 🎯 Learning Objectives
- [x] Master **Model Binding** for form processing.
- [x] Implement **Data Annotations** for server-side validation.
- [x] Use **Razor Tag Helpers** (`asp-for`, `asp-validation-summary`) to build dynamic forms.
- [x] Practice the **Post-Redirect-Get (PRG)** pattern for robust form submissions.

## 📂 Folder Structure
- [**`/src`**](./src/README.md): Detailed explanation of the RSVP implementation, GuestResponse model, and in-memory repository.
- [**`/tests`**](./tests/README.md): Automated unit tests verifying the validation rules and controller logic.

## 🛠️ Key Technologies
- **Framework**: .NET 10
- **App Type**: ASP.NET Core MVC
- **Quality**: xUnit + Strongly-typed Validation

---
> [!IMPORTANT]
> This unit bridges the gap between simple page rendering and active user interaction. Mastering form handling and validation is essential for building real-world business applications.
