# 🚀 Unit 02: Hello MVC

Welcome to the second unit. Now that the foundation is set, we dive into the core **Model-View-Controller (MVC)** architectural pattern.

## 🎯 Learning Objectives
- [x] Understand the **MVC Request Life Cycle**.
- [x] Implement **Controllers** and **Action Results**.
- [x] Create dynamic **Models** and **Views**.
- [x] Master **Routing** and **Tag Helpers**.

## 📂 Folder Structure
- [**`/src`**](./src/README.md): Detailed explanation of the MVC implementation and controllers.
- [**`/tests`**](./tests/README.md): Unit tests for the `HomeController` logic.

## 🛠️ Key Technologies
- **Framework**: .NET 10
- **App Type**: ASP.NET Core MVC
- **Pattern**: Separation of Concerns (SoC)
